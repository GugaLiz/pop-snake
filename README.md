# 消消蛇

纯前端网页小游戏，融合贪吃蛇走位、蛇尾颜色三连消除、短局强化、每日挑战、方向解谜和箭蛇破阵。

## 当前状态

- V3 入口已收敛为：主玩法、玩法分支、更多玩法。
- 无需后端接口即可完整游玩。
- 支持 PC 和移动端浏览器。
- 支持方向键、WASD、滑动和移动端虚拟方向键。
- 支持本地最高分、最长生存时间、最高 Combo、设置项、每日状态、贴纸和连续达成记录保存。
- 支持基础音效、震屏开关、消除粒子、Combo 浮字、新手提示和模式结果页。

## 模式

- 冲刺模式：90 秒短局冲分，三消返时，每完成 2 次三消可选择 1 个局内强化。
- 大乱斗：120 秒展示赛模式，随机连续闯 5 个小关，混合冲刺、解谜、破阵和染色玩法。
- 每日挑战：每天固定规则和目标，当前包含 6 种每日规则，并可解锁每日贴纸、记录连续达成。
- 箭头解谜蛇：10 关首章，必须用正确方向进入目标格，普通色块可辅助补尾和三消，清空方向目标即可过关。
- 箭蛇破阵：45 秒破阵玩法，金色核心被围墙包住，至少吃 2 个核心并达到目标分才算成功；按空格或移动端射击按钮击穿围墙，吃色块三消可给下一次射击附魔。
- 同色匹配蛇：75 秒控色玩法，每次有效转向后蛇头按固定色序切换，只能吃同色食物。
- 5秒同色蛇：75 秒控色玩法，蛇头每 5 秒自动按固定色序切换，只能吃当前同色食物。
- 更多玩法：保留无尽、精准、标准、限时、限步等旧模式作为补充玩法。

## 核心规则

- 吃到普通色块后，蛇尾新增对应颜色的一节。
- 每次吃块后检查蛇尾最后 3 节。
- 最后 3 节同色时，消除这 3 节并得分。
- 彩虹节可作为任意颜色参与蛇尾三连。
- 炸弹色块会直接消除蛇尾。
- 减速色块会短时间降低移动速度。
- 冲刺和每日模式会根据三消、Combo、任务和强化产生额外返时或得分。
- 箭蛇破阵模式中，核心默认被围墙包围，必须先射击击穿墙壁再冲进去吃核心；吃到核心可得分、加时并进入下一波，三消会给下一次射击附加破阵效果。
- 染色类模式中，蛇头颜色按固定顺序循环，方向染色由转向触发，5秒变色由时间触发；当前色只能吃同色食物，吃对后尾巴加入对应颜色。
