# 消消蛇

纯前端网页小游戏，融合贪吃蛇走位和蛇尾颜色三连消除。

## 当前功能

- 无需后端接口。
- 支持 PC 和移动端浏览器。
- 支持方向键、WASD、滑动和移动端虚拟方向键。
- 支持 5 种模式：标准、无尽、限时、限步、精准。
- 支持 3 种特殊色块：炸弹、彩虹、减速。
- 支持本地最高分、最长生存时间、最高 Combo、设置项保存。
- 支持基础音效、吃块吸附、消除粒子、Combo 浮字、新手提示。

## 安装

```powershell
npm install
```

## 开发运行

```powershell
npm run dev
```

打开终端显示的本地地址，例如 `http://localhost:5173/`。

## 构建

```powershell
npm run build
```

## 玩法规则

- 吃到普通色块后，蛇尾新增对应颜色的一节。
- 每次吃块后检查蛇尾最后 3 节。
- 最后 3 节同色时，消除这 3 节并得分。
- 彩虹节可作为任意颜色参与蛇尾三连。
- 炸弹色块会直接消除蛇尾 3 节。
- 减速色块会短时间降低移动速度。

## 模式

- 标准模式：有边界限制，达到目标分数即可过关。`r`n- 无尽模式：无边界限制，穿墙循环，只在撞到身体时结束。
- 限时模式：60 秒内达到目标分数或目标消除数。
- 限步模式：限定步数内达到目标消除数。
- 精准模式：限定步数结束时蛇长刚好达到目标值。
